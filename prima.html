About PRIMA

Methodology and design principles.

PRIMA is a cognitive training simulator designed to install the thinking patterns used by Elon Musk and other first-principles operators. It is not a biography. It is a workout.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality. Forgetting curve combat.
  • Active Recall: Every drill requires generation, not recognition. You must produce the answer, not just identify it.
  • Interleaving: Drills mix concepts across modules. Forced discrimination strengthens memory vs. blocked practice.
  • Elaboration: Arena problems and Project Prometheus require applying concepts to novel scenarios.

// The Ten Axes

  • AXIOM — First principles as default reasoning mode
  • INDEX — The idiot index; cost decomposition
  • NEGATE — Questioning requirements; "the best part is no part"
  • FERMI — Physics-based order-of-magnitude reasoning
  • ASYMM — Asymmetric bets and expected value
  • VELOC — Execution velocity as force multiplier
  • LOOP — Closing feedback loops with reality
  • WEAVE — Cross-domain transfer of mental models
  • ALGO — The 5-step engineering algorithm
  • PSYCH — Psychological architecture & pain tolerance
