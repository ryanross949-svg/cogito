About DIJKSTRA

Methodology and design principles.

DIJKSTRA is a cognitive training simulator designed to install the algorithmic thinking patterns used by computer scientists and software engineers. It trains you to break down complex goals into optimal, step-by-step processes, finding the shortest path through any maze.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Nexus require applying concepts to novel scenarios.

// The Ten Axes

  • GRAPH — Graph traversal, BFS, DFS, adjacency lists
  • SHORTEST — Dijkstra's algorithm, weighted graphs, edge relaxation
  • BIGO — Time and space complexity, asymptotic notation
  • SORT — Comparison sorts, O(n log n) limits, stability
  • SEARCH — Binary search, divide and conquer
  • DP — Dynamic programming, overlapping subproblems, memoization
  • GREEDY — Local optimum vs global, MSTs
  • HEURISTIC — A* search, admissibility, informed search
  • TREE — Binary search trees, traversals, balance
  • PATTERN — Two pointers, sliding window, string matching
