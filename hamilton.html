About HAMILTON

Methodology and design principles.

HAMILTON is a cognitive training simulator designed to install the biological thinking patterns used by evolutionary biologists and geneticists. It is not a textbook. It is a workout.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality. Forgetting curve combat.
  • Active Recall: Every drill requires generation, not recognition. You must produce the answer, not just identify it.
  • Interleaving: Drills mix concepts across modules. Forced discrimination strengthens memory vs. blocked practice.
  • Elaboration: Arena problems and Project GENESIS require applying concepts to novel scenarios.
