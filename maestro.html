About MAESTRO

Methodology and design principles.

MAESTRO is a cognitive training simulator designed to install the systems thinking patterns used by top-tier architects, infrastructure engineers, and complexity theorists. It trains you to see the invisible wires connecting everything.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Architect require applying concepts to novel scenarios.

// The Ten Axes

  • SYSTEM — Elements, interconnections, purpose/function
  • FLOW — Stocks, flows, rates of change
  • LOOP — Positive and negative feedback
  • BOTTLENECK — Theory of Constraints
  • SCALE — Scaling laws (Kleiber, Metcalfe)
  • EMERGE — Complex adaptive systems, phase transitions
  • RESILIENCE — Robustness vs fragility vs antifragility
  • ENTROPY — Software rot, technical debt, 2nd law
  • INTERFACE — APIs, Conway's Law, coupling
  • TRADE — Multi-objective optimization, Pareto efficiency
