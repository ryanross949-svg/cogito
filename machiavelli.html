About MACHIAVELLI

A field manual for the dark arts — built for defense.

MACHIAVELLI is a hidden cognitive training simulator that maps the architecture of influence and manipulation — reciprocity, commitment, social proof, authority, liking, scarcity, fear, the Dark Triad, and covert language. It is taught from the defenders chair: you learn exactly how each lever is pulled so that it can never be pulled on you.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix tactics across modules.
  • Elaboration: Arena problems and Project Iago require spotting and countering manipulation in novel scenarios.

// The Ten Axes

  • RECIPROCITY — Manufactured debt, free samples, door-in-the-face
  • COMMITMENT — Foot-in-the-door, public commitments, sunk cost
  • PROOF — Manufactured consensus, fake reviews, bandwagons
  • AUTHORITY — Milgram, titles, uniforms, the expert frame
  • LIKING — Mirroring, flattery, the halo effect
  • SCARCITY — Urgency, FOMO, exclusivity, loss aversion
  • FOG — Fear, obligation, guilt, gaslighting, trauma bonds
  • TRIAD — Narcissism, Machiavellianism, psychopathy
  • LANGUAGE — Framing, presupposition, false dichotomy, anchoring
  • DEFENSE — The pause, naming tactics, boundaries, gray rock

This material is provided for awareness and self-defense. Recognizing manipulation is the surest protection against it.
