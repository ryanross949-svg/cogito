Topology Lab

A Mobius strip: a surface with only one side and one edge. Trace it and you return to the start having visited "both" faces. The Klein bottle is its closed cousin — a bottle whose inside is its outside.

About KLEIN

Methodology and design principles.

KLEIN is a cognitive training simulator designed to install the spatial reasoning patterns used by topologists and pure mathematicians. It trains you to identify properties of objects that remain invariant even when the object is stretched, twisted, or deformed, breaking rigid Euclidean thinking.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Eversion require applying concepts to novel scenarios.

// The Ten Axes

  • MOBIUS — The Möbius strip, one-sided surfaces, non-orientability
  • DEFORM — Rubber-sheet geometry, homeomorphism, topological equivalence
  • BOUND — Boundaries, manifolds with and without edges
  • HOLES — Genus, Euler characteristic, counting holes
  • KNOTS — Knot theory, unknots, trefoils, invariants
  • INSIDE — Jordan curve theorem, inside vs outside
  • CONT — Connectedness, path-connectedness, gluing
  • FIXED — Fixed point theorems, Brouwer, Hairy Ball
  • DIMENS — Higher dimensions, 4D spaces, tesseracts
  • KLEIN — The Klein bottle, closed non-orientable surfaces
