Memory Lab

The method of loci turns memory into a walk. Each item is placed at a familiar spot along a route; to recall, you stroll the route and collect them. Study the five placements, then take the walk and see how many you pin to the right spot.

About LOCI

Methodology and design principles.

LOCI is a cognitive training simulator designed to install the mnemonic techniques used by memory athletes. It trains you to artificially expand working memory using spatial navigation, chunking, and associative imagery.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Mnemonic Vault require applying concepts to novel scenarios.

// The Ten Axes

  • PALACE — The Method of Loci (Memory Palaces)
  • CHUNK — Working memory limits (7±2) and chunking
  • MAJOR — The Major System (numbers to consonants)
  • LINK — The Linking Method & Story Method
  • PEG — The Peg System (rhyming/shape pegs)
  • SPACING — The Spacing Effect & Ebbinghaus curve
  • VISUAL — Visualization & Vivid Imagery
  • ASSOC — Associative Hooks & Elaboration
  • REVIEW — Active Recall & Testing Effect
  • OVERL — Overlearning & Automaticity
