About VON NEUMANN

Methodology and design principles.

VON NEUMANN is a cognitive training simulator designed to install the architectural thinking patterns used by John von Neumann and modern computer scientists. It bridges logic (Turing) and biology (Mendel).

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project PROBE require applying concepts to novel scenarios.
