// How Cogito Works

What this is

Cogito is a gamified learning platform built on spaced repetition. Each course distills a field into 10 modules of flashcards, applied problems, and a capstone project — across 7 disciplines: Physics, Mathematics, Computation, Life, Mind, Strategy, and Systems.

The Hub (this page)

  • Courses are grouped by discipline — click a discipline header to collapse it.
  • Your Courses: tap the ★ star on any course to pin it to the top strip. Nothing is added automatically.
  • Your Path: take the Find My Starting Point quiz for a recommendation, then work the auto-generated to-do list — each task deep-links to the exact module, Arena, or Drills.
  • Daily Review banner: once you finish a course’s first full pass, its due cards join a single cross-course review queue.
  • Global Mastery Matrix & radar: your mastery across every discipline at a glance.
  • Themes: 8 color themes. Achievements & Shop: earn and spend as you progress.

Inside a course

  • Dashboard — overview and your daily mission.
  • Modules — read each lesson, then unlock the next by studying its cards.
  • Drills — the spaced-repetition flashcards (see below).
  • Arena — applied multiple-choice problems for bonus XP.
  • Project — a multi-phase capstone that ties the course together.
  • Stats — retention, streaks, and progress.

How spaced repetition works

In Drills, read the prompt, reveal the answer, then rate yourself honestly:

  • Again — forgot it; the card returns this session.
  • Hard / Good / Easy — the gap until you see it again grows each time (Good runs roughly 3d → 7d → 18d → 45d…), so you review each fact right before you’d forget it.
  • Use the order toggle by the filters to run cards in Module order or Shuffle.

Shortcuts & tips

  • In Drills: space reveals, 1–4 rate Again / Hard / Good / Easy.
  • Pin your active courses with the ★ so they’re always one tap away.
  • Follow the Recommended tasks for the highest-leverage next action at any moment.

Install as an app

Cogito runs in the browser, but you can add it to your phone so it behaves like a native app — full screen, its own icon on the home screen, no address bar. Your progress is saved on the device and carries straight over.

  • iPhone / iPad (Safari): tap the Share button (the square with an up-arrow), scroll down, and tap Add to Home Screen.
  • Android (Chrome): tap the ⋮ menu at the top-right, then Add to Home screen (or Install app).
  • Desktop (Chrome / Edge): click the install icon in the address bar, or the menu → Install.

Then launch Cogito from its new icon, just like any other app.

Reset

This wipes everything on this device — XP, cards, streaks, unlocks, themes, and pinned courses. It cannot be undone.
