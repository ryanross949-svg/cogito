About DANTZIG

Methodology and design principles.

DANTZIG is a cognitive training simulator designed to install the optimization thinking patterns used by operations researchers and linear programmers. It trains you to find the absolute mathematical maximum (profit, speed, output) when bound by finite constraints.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Apex require applying concepts to novel scenarios.

// The Ten Axes

  • FEASIBLE — The feasible region (graphing constraints, corner points)
  • OBJECTIVE — Objective functions (maximize/minimize)
  • SIMPLEX — The Simplex method (pivoting, basic variables)
  • DUALITY — Primal-dual relationships (shadow prices)
  • BINDING — Binding vs non-binding constraints (slack)
  • SHADOW — Shadow prices and reduced costs (marginal value)
  • NETWORK — Network flow and transportation (max flow, min cut)
  • INTEGER — Integer programming (branch and bound, binary vars)
  • SENSITIVITY — Sensitivity analysis (ranges of optimality)
  • SCARCITY — Optimization under scarcity (resource allocation)
