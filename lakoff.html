About LAKOFF

Methodology and design principles.

LAKOFF is a cognitive training simulator designed to install the thinking patterns used by cognitive linguists, rhetoricians, and persuasion experts. It is not a grammar textbook. It is a workout.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality. Forgetting curve combat.
  • Active Recall: Every drill requires generation, not recognition. You must produce the answer, not just identify it.
  • Interleaving: Drills mix concepts across modules. Forced discrimination strengthens memory vs. blocked practice.
  • Elaboration: Arena problems and Project METAPHOR require applying concepts to novel scenarios.

// The Ten Axes

  • EMBODIED — Embodied cognition
  • CONCEPT — Conceptual metaphors
  • FRAME — Framing effects
  • ORIENT — Orientational metaphors
  • ONTO — Ontological metaphors
