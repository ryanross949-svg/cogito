About CONWAY

Methodology and design principles.

CONWAY is a cognitive training simulator designed to install the thinking patterns of bottom-up, decentralized computation. It explores how complex, lifelike behavior emerges from grids of dead/alive cells following simple rules.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Autopoiesis require applying concepts to novel scenarios.

// The Ten Axes

  • AXIOM — The Grid & Rules (Conway's Game of Life, binary states)
  • MOTION — Gliders & Spaceships (Moving patterns, speed limits)
  • COMPUTE — Turing Completeness (Logic gates, universal computation)
  • STILL — Still Lifes & Oscillators (Stable states, periodicity)
  • METHUSELAH — Long-lived patterns (Evolution, stabilization, entropy)
  • REPLICATE — Self-Replication (Loops, universal constructors)
  • RULE — Cellular Automata Theory (Rule 110, Wolfram classes)
  • SCALE — Digital Ecology (Predator-prey, flocking, swarms)
  • EQUILIB — Chaos and Stability (Attractors, edge of chaos)
  • DECODE — Decentralized Computation (Bottom-up design, biology)
