About BAYES

Methodology and design principles.

BAYES is a cognitive training simulator designed to install the mathematical engine of rationality. It trains you to update your beliefs correctly under uncertainty, counteract cognitive biases with exact formulas, and navigate the false positive paradox.

// Core Pedagogy

The system uses four evidence-based learning mechanisms:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals.
  • Active Recall: Every drill requires generation.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems require applying formulas to novel scenarios.

// The Ten Axes

  • PRIOR — Base rates, default assumptions, P(A)
  • EVIDENCE — Likelihood, sensitivity, specificity, P(B|A)
  • UPDATE — Bayes' Theorem formula, posterior probability
  • CALIBRATE — False positives, base rate neglect, practical application
  • ODDS — Odds form of Bayes, log-odds, weights of evidence
  • DIST — Normal distributions, variance, tail risk, CLT
  • EV — Expected value, St. Petersburg paradox, Kelly criterion
  • SIGNAL — Signal-to-noise ratio, evidence independence
  • MODEL — Model selection, Occam's razor, Bayesian networks
  • INFO — Shannon entropy, information gain, uncertainty
