About LORENZ

Methodology and design principles.

LORENZ is a cognitive training simulator designed to install the strategic thinking patterns required to navigate chaos, complexity, and non-linear systems. It trains you to recognize when deterministic prediction is impossible and how to build robust systems that survive the Butterfly Effect.

// Core Pedagogy

The system uses four evidence-based learning mechanisms, applied in concert:

  • Spaced Repetition (SM-2): Cards reappear at expanding intervals based on recall quality.
  • Active Recall: Every drill requires generation, not recognition.
  • Interleaving: Drills mix concepts across modules.
  • Elaboration: Arena problems and Project Strange require applying concepts to novel scenarios.

// The Ten Axes

  • SENSITIVITY — The Butterfly Effect, initial conditions
  • NONLINEAR — Non-linear dynamics, disproportionality
  • ATTRACT — Strange attractors, bounded chaos
  • BIFURC — Bifurcation, period doubling, Feigenbaum
  • FRACTAL — Fractal geometry, self-similarity, non-integer dimensions
  • PHASE — Phase space, trajectories, topological mixing
  • FEEDBACK — Positive/negative loops driving non-linear behavior
  • PREDICT — Predictability horizon, Lyapunov time, ensemble forecasting
  • EMERGE — Edge of chaos, criticality, self-organization
  • NAVIGATE — Robustness vs prediction, exploiting chaos
